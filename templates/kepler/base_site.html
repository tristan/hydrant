Welcome to Hydrant

What is this?

This project is intended to create a Web portal for executing kepler workflows.

The project is currently still in the proof of concept stage

You can log in to play with things using either of the following logins
Username: designer, Password: test
Username: scientist, Password: test

The designer user reflects the type of user who is kepler savvy and would be responsible for building and deploying workflows for the scientist. The designer can upload workflows and set up exposed parameters, and other properties relevent to the workflow to make it easy for scientists to use.

The scientist user reflects the type of user who just wants to run workflows and doesn't want to have to learn how kepler works or how to use it. The scientist only has access to select and execute avaliable workflows.

Best viewed using Firefox >= 2.0 (i.e. looks terrible, and doesn't work right in other browsers. Compatibility is high on the to-do list)

Development info availiable on the kepler @ jcu trac
