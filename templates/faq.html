Why wont my workflow upload?

If you get a Workflow Validation error when attempting to upload a workflow, it means that the XML validation process which checks every workflow when it's uploaded failed. Check to make sure that what you are uploading is a valid MoML document (if it load's in kepler it should be fine).

I can see a Job, but I can't see the workflow it is created from. What's going on?

There are two reason's this may happen. The first being that the workflow in question may have been deleted since the job was run. The other is related to workflow and job permissions. Workflow and Job permissions are seperate from each other. This means that a user can share a job with you which has been created from a workflow which you don't have access to. You can try sending a message to the owner of the Job and see whether they will make the workflow visable to you.
